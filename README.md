ceph-cookbook

Cookbook for deploying ceph cluster.

------------

Usage
-----
#### ceph-cookbook::default

e.g.
Just include `ceph-cookbook` in your node's `run_list`:

```bash
mb ceph-deploy bootstrap bootstrap.json --environment ceph_cluster -v
```
